# HTML Resume Website

A simple single-page resume website built using pure HTML.  
This project demonstrates basic HTML structure, semantic elements, lists, and tables without using CSS.

## Features

- Header with name and title
- About section
- Skills list
- Experience table
- Projects section
- Education table
- Contact details
- Footer

## Setup

Open `index.html` in any web browser.
No installation required.

## Usage

Edit the HTML file to customize the resume with your own details.
